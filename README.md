# ChatPDC
O Chat com as respostas mais precisas e certas do mundo

Para mandar mensagem só digitar e dar enter

Comandos:
HELP - Show all commands
CLEAR - Clean the terminal
QUIT - Quit ChatPCD
SELECT - Change ChatPDC Mode

Modos:
1. Normal: Preciso e acertivo
2. Curto e grosso: Breve e cirurgico nas respostas
3. Filósofo: Mais rebuscado e culto
4. Listador: Ele adora listar coisas

KoC Trigger:
É um sistema que tem 1% de chance de te dar uma resposta diretamente de diálogos profundos com o King of Cows
